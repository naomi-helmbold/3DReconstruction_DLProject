# 3D Reconstruction Deep Learning Project

A deep learning project focused on 3D reconstruction of a specific scene (the Amigo Room in Ecole Centrale de Lyon) using Neural Radiance Fields (NeRF).

## Project Overview

This project aims to reimplement the Neural Radiance Fields (NeRF) method for 3D reconstruction and train and apply it to a specific use case, of reconstructing a experimentation room from the Ecole Centrale de Lyon campus .

## Current Status

This project is in its first complete version, with the following steps complete:

- ✅ Initial project setup and structure
- ✅ Training data collection and organization
- ✅ Training images and poses for model development (Blender rendering and actual training images)
- ✅ Data preprocessing and structured codebase implementation
- ✅ Model development and training (in progress)
